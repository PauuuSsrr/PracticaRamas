# Història de Git

Git és un sistema de control de versions distribuït creat per Linus Torvalds l'any 2005.

Va ser desenvolupat per gestionar el codi font del nucli de Linux de manera eficient, ràpida i segura.

Actualment, Git és utilitzat a tot el món per equips de desenvolupament per controlar canvis, col·laborar i mantenir l'historial dels projectes.
